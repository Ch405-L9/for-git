# PROMPT 2: Dark Minimal Tech - Web Optimization Service

**Mission:** Build landing page for web optimization agency with dark theme, tech animations, modern minimalist aesthetic.

## Design Specifications

**Color Palette:**
```
Background: #0A0A0A (near-black)
Surface: #1A1A1A (cards)
Border: #2A2A2A (subtle lines)
Primary: #00D9FF (electric cyan)
Accent: #7B61FF (violet)
Text Primary: #FFFFFF
Text Secondary: #A0A0A0
Success: #00FF88 (for metrics)
```

**Typography:**
- Headings: Inter (700, 800 weights)
- Body: Inter (400, 500 weights)
- Code/metrics: JetBrains Mono
- Sizing: Ultra-large hero (text-7xl → text-9xl), crisp body (16-18px)

**Hero Section Design:**

*Layout:*
- Split screen: Left (content), Right (animated visual)
- Vertical center alignment
- Max-width: 1400px container

*Content (Left):*
- Eyebrow text: "PERFORMANCE ENGINEERING" (12px, tracking-widest, cyan)
- H1: "Lightning-Fast Websites That Convert" (96px desktop, gradient text cyan→violet)
- Description: "Core Web Vitals optimization, CDN implementation, and speed audits that boost rankings and revenue" (20px, gray-400)
- Dual CTAs: "Analyze My Site" (primary cyan) + "View Case Studies" (outline)
- Trust indicators: "98 AVGSPEED SCORE • 2.1s AVG LOAD • $4.2M REVENUE IMPACT" (monospace, small, success green)

*Visual (Right):*
- Floating dashboard mockup with live metrics
- Animated number counters
- Pulse effects on metric cards
- Glowing cyan border + shadow

**Animated Service Cards:**

*Card Structure (3-column grid):*
```
┌─────────────────┐
│ [Icon]          │ ← Animated icon (rotate on hover)
│ Title           │
│ Description     │
│ • Benefit 1     │
│ • Benefit 2     │
│ • Benefit 3     │
│ [Learn More →]  │
└─────────────────┘
```

*Animations Required:*

1. **Hover State:**
   - Border glow (cyan, 2px, blur 8px)
   - Translate Y: -8px
   - Scale: 1.02
   - Icon rotates 5deg

2. **Entrance:**
   - Stagger delay (0.1s between cards)
   - Fade + slide from bottom
   - Duration: 0.6s ease-out

3. **Background Effect:**
   - Gradient mesh moves on mouse position
   - Particle grid in background
   - Parallax scroll (cards move slower than scroll)

**Services to Feature:**

1. **Core Web Vitals Optimization**
   - Icon: Lightning bolt
   - Metrics: LCP, FID, CLS improvements
   - Tech: Next.js, image optimization, lazy loading

2. **CDN & Edge Computing**
   - Icon: Network nodes
   - Global distribution, edge caching
   - Tech: Cloudflare, Vercel Edge

3. **Speed Audits & Monitoring**
   - Icon: Speedometer
   - Lighthouse CI, RUM tracking
   - Monthly performance reports

**Tech Animation Ideas:**

*Background:*
- Dot grid pattern (subtle, #1A1A1A)
- Animated gradient orbs (blur 100px, moving slowly)
- Scan line effect (subtle horizontal line moving down)

*Interactive Elements:*
- Code snippets that type out
- Live Lighthouse score animation
- Before/After load time comparison (animated bars)
- Network waterfall visualization

*Micro-interactions:*
- Button hover: glow effect + arrow slide
- Link underlines: gradient wipe animation
- Form inputs: neon focus ring (cyan)
- Scroll indicator: animated chevron

*CTA Section:*

**Technical Implementation Notes:**

*Animations Library:*
- Framer Motion for React components
- CSS custom properties for theme
- Intersection Observer for scroll triggers

*Performance:*
- Lazy load all animations below fold
- Use `will-change` sparingly
- GPU-accelerated transforms only
- Code-split animation library

*Accessibility:*
- `prefers-reduced-motion` media query (disable animations)
- High contrast mode support
- Keyboard navigation for all interactions
- ARIA labels on interactive elements
