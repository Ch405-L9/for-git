Dark Mode Design
Ideal Use Case: Developer Tools, Communities
Purpose & Feel: Sleek efficiency + eyes‑off contrast
Example: github.com

Prompt
text
Build a DARK‑MODE‑FIRST application shell.
- Palette #0d1117 background, #c9d1d9 text.
- Accent #58a6ff links; syntax green #3fb950.
- Components: Navbar → Editor → Output Pane (+ Toggle theme light/dark).
- Ensure prefers‑color‑scheme CSS media query.
Reference CSS
css
@media(prefers-color-scheme:dark){
  body{background:#0d1117;color:#c9d1d9;}
  a{color:#58a6ff;}
  code{color:#3fb950;}
}
Editing Prompts
“Invert theme colors for Light Mode.”

“Enable auto‑switch based on system preference.”

“Add high‑contrast mode with yellow alerts.”

Retro / Futuristic
Ideal Use Case: Music, Gaming, Startups
Purpose & Feel: Nostalgic neon energy
Example: cyberpunk.net

Prompt
text
Design a RETRO‑FUTURIST site using cyberpunk palette:
- Background black #000 gradients to neon magenta → cyan.
- Typography: pixel or sci‑fi fonts like Orbitron.
- Animated scanlines/ambient glow FX.
- Audio‑visual loop (optional) < 2MB.
Reference CSS
css
:root{
  --magenta:#ff00c8;--cyan:#00fff7;
}
h1{
  font-family:'Orbitron',sans-serif;
  text-shadow:0 0 10px var(--magenta),0 0 20px var(--cyan);
}
Editing Prompts
“Animate gradient background 5 s linear infinite.”

“Add flicker effect to hero text.”

“Switch neon palette to orange/teal.”

Organic / Natural Design
Ideal Use Case: Wellness, Non‑profit, Eco brands
Purpose & Feel: Human‑centered calm
Example: etsy.com

Prompt
text
Generate an ORGANIC‑THEMED website.
- Palette: earth tones #faf3e0, #decdb0, #7a7052.
- Rounded corners, soft drop shadows < blur 8 px.
- Typography: serif headlines + open sans body.
- Include leaf SVG motifs + gentle fade‑in animation.
Illustrative Design

Example: dropbox.com

Prompt
text
Create a HAND‑DRAWN illustration‑first layout.

3D / Interactive Design
Ideal Use Case: Media, Product Demos
Purpose & Feel: Immersive presentation
Example: apple.com/mac-pro/

Prompt
text
Integrate three.js or Spline scene into Hero component.

### Skeuomorphic Design
Ideal Use Case: Retro, Learning, Simulations
Purpose & Feel: Familiar realism
Example: 99designs.com/blog/trends/skeuomorphism

#### Prompt

text
Recreate real‑world controls with CSS textures.
- Use embossed/debossed buttons with multi‑layer shadows.
- Add light reflections linear‑gradient(top to transparent 25%).
- Must remain accessible (focus ring visible > 3:1 contrast).
